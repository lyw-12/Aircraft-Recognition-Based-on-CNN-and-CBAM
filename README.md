# 基于CNN与CBAM注意力机制的飞行器目标识别系统

## 项目简介

本项目基于 PyTorch 实现飞行器目标识别系统，旨在研究卷积神经网络（CNN）在飞行器图像分类任务中的应用效果。项目构建了包含军用飞机、民航客机、无人机和直升机四类目标的自定义数据集，并完成图像预处理、数据增强、模型训练与性能评估等工作。

在基础CNN模型的基础上，引入残差连接（Residual Block）和卷积块注意力模块（CBAM），有效提升了模型特征提取能力和分类性能，同时实现模型轻量化设计。

---

## 项目特点

* 自建飞行器图像数据集
* 基于CNN的图像分类模型
* 支持数据增强与标准化预处理
* 引入残差连接优化深层网络训练
* 融合CBAM注意力机制增强特征表达
* 支持CUDA/cuDNN GPU加速训练
* 完整的训练、验证与测试流程
* 自动保存模型权重与训练日志

| 模型                    | 准确率  | 参数量    |
