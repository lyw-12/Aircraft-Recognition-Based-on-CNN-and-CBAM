基于深度学习卷积神经网络的飞行器目标识别系统

本项目基于 PyTorch 实现了一套飞行器目标识别系统，构建了包含军用飞机、民航客机、无人机和直升机四类目标的自定义数据集。项目设计了基础 CNN 模型，并在此基础上引入残差连接（Residual Block）和卷积块注意力模块（CBAM），实现了模型性能与轻量化的同步提升。

项目特点：

自建飞行器图像数据集
基于 CNN 的飞行器目标分类
数据增强与标准化预处理
引入 ResNet 残差结构优化训练过程
融合 CBAM 注意力机制增强特征提取能力
支持 GPU（CUDA/cuDNN）加速训练
实现模型训练、验证、测试完整流程

实验结果：

基础 CNN 模型准确率：92%+
改进模型准确率：97%+
参数量由约 5MB 降低至 2.8MB
模型体积减少超过 40%

本项目为高精度、轻量化飞行器目标识别提供了一种有效解决方案，可应用于军事侦察、航空安全、无人机监管及智能监控等场景。
